## Qué editar primero
### 1) Links de Mercado Pago
Buscá estas 4 cadenas en `index.html` y pegá un link distinto en cada una:
- `REEMPLAZAR_LINK_MP_1000`
- `REEMPLAZAR_LINK_MP_5000`
- `REEMPLAZAR_LINK_MP_10000`
- `REEMPLAZAR_LINK_MP_OTRO_MONTO`

### 2) Alias
Ya quedó cargado:
- `michelf.97`

### 3) Video principal
Reemplazá:
- `assets/hero-video.mp4`

Si querés mostrar una miniatura antes de cargar el video, reemplazá:
- `assets/hero-poster.jpg`

## Abrir local
Podés abrir `index.html` directamente o usar Live Server en VS Code.

## Deploy en Vercel
1. Subí esta carpeta a GitHub
2. Importá el repo en Vercel
3. Deploy directo, sin configuración extra
